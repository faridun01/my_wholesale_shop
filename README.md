# My Wholesale Shop

Wholesale CRM and inventory system with:
- warehouse management
- products and stock tracking
- POS terminal
- sales history
- customers and reminders
- reports

## Run Locally

Prerequisites: Node.js, PostgreSQL

1. Install dependencies:
   `npm install`
2. Configure backend environment variables in `backend/.env`
3. Start the backend:
   `npm run dev:backend`
4. Start the frontend:
   `npm run dev:frontend`

Or run both together:
`npm run dev:all`
